\#  Green Studio



Full Stack Photography Studio Web Application



\## Tech Stack

\### Frontend

\- React.js

\- Tailwind CSS

\- React Router



\### Backend

\- Spring Boot

\- MySQL

\- JWT Authentication

\- Spring Security



\## Deployment

\- Frontend → Netlify

\- Backend  → Render

\- Database → Clever Cloud MySQL



\## Features

\- User Registration \& Login

\- Event Booking System

\- Admin Dashboard

\- JWT Authentication

\- Responsive Design
